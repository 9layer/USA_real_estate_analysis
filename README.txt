## USA Real Estate Analysis

### Описание проекта

Проект предназначен для анализа рынка недвижимости в США. Он включает сбор и обработку данных о недвижимости, визуализацию ключевых метрик и географический анализ активности по штатам и городам. Проект демонстрирует навыки работы с Python, Pandas, визуализацией и обработкой больших данных для аналитики.
